# Paradise Nursery - Houseplant Shopping App

A React-based e-commerce application for buying houseplants with shopping cart functionality.

## Features
- **Landing Page** with company information and Get Started button
- **Product Listing Page** with categorized houseplants
- **Shopping Cart Page** with full CRUD operations
- **Redux State Management** for cart functionality
- **Responsive Design** for all screen sizes

## Pages
1. **Home** - Landing page with company overview
2. **Products** - Browse houseplants by categories
3. **Cart** - Manage shopping cart items

## Technologies Used
- React 18
- Vite
- Redux Toolkit
- React Router DOM
- Lucide React Icons
